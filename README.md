# 🛍️ StyleHub – React E-Commerce Application

## 🚀 Project Description
StyleHub is a responsive e-commerce web application built using React.  
The application allows users to manage products with full CRUD functionality (Create, Read, Update, Delete).  
It demonstrates component-based architecture and modern frontend development practices.

---

## 🛠️ Tech Stack
- React.js
- JavaScript (ES6)
- HTML5
- CSS3
- JSON Server (Mock Backend)

---

## ✨ Key Features
- 📦 Product listing
- ➕ Add new products
- ✏️ Edit product details
- ❌ Delete products
- 🔄 Dynamic rendering using React components
- 📱 Responsive user interface
- ⚛️ React Hooks (useState, useEffect)

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/swapnareddy18/stylehub-react.git
```

Navigate into project folder:

```bash
cd stylehub-react
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm start
```
